# Java-Bee-Simulation 🐝
### Various implementations of population-based search algorithms related to bees and their foraging relationship with flowers.
- A collaborative university project in a team of 3
- Leverages numerous programming paradigms and design patterns such as:<br>
     * Procedural, Functional (Lambdas and Streams), Object-Oriented, Parallel programming
     * Factory Method, Abstract Factory, Visitor, "Design by Contract" patterns
